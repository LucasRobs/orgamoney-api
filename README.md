# Orgamoney

---
## How to install
```sh
git clone https://github.com/LucasRobs/orgamoney-api
cd orgamoney-api
npm install
```
---
## Settings
create an .env file
put the following content with your database information
```env
DATABASE_URL=[your bank name]://[login]:[password]@localhost:5432/[bank name]
```
and run
```sh
npm run dev
```
